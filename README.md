## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/dragonpilee/digital-image-processing.git
   cd digital-image-processing
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run any script**
   ```bash
   python script_name.py
   ```
